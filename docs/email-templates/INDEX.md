# Tekmadev auth email templates

Generated by `build.mjs`. To use: in Supabase go to **Authentication > Emails > Templates**, open each template, set its **Subject**, and paste the file contents into the **Message body** (source/HTML).

| Supabase template | File | Subject |
|---|---|---|
| Confirm sign up | `confirm-signup.html` | Confirm your email for Tekmadev |
| Invite user | `invite.html` | You've been invited to Tekmadev |
| Magic link or OTP | `magic-link.html` | Your Tekmadev sign-in link |
| Change email address | `change-email.html` | Confirm your new email address |
| Reset password | `reset-password.html` | Reset your Tekmadev password |
| Reauthentication | `reauthentication.html` | Your Tekmadev verification code |

Variables used: `{{ .ConfirmationURL }}`, `{{ .Token }}`, `{{ .Email }}`, `{{ .NewEmail }}`.
`_preview-reset.html` is a sample-filled preview only, do not paste it into Supabase.
